## Install

### Linux / macOS / WSL2 / Android (Termux)

```bash
curl -fsSL https://tj-coding.github.io/nautilus-community/install.sh | bash
```

### Windows (native)

Run in PowerShell:

```powershell
iex (irm https://tj-coding.github.io/nautilus-community/install.ps1)
```

After installing, run **nautilus setup** to configure your provider, tools, and platform.

:::tip Fastest path to a working agent
Run `nautilus setup --portal` — one OAuth covers a model plus all Tool Gateway tools (web search, image generation, TTS, browser). See [Nous Portal](/integrations/nous-portal).
:::

## What is Nautilus Agent?

Nautilus takes everything that makes [Hermes Agent](https://github.com/NousResearch/hermes-agent) the most capable open-source agent framework — built-in learning loop, skills system, cron scheduler, subagents, and cross-platform gateway — and adds a **science layer**:

| Science Capability | What you can do |
|-------------------|-----------------|
| **Protocol Automation** | Turn experimental protocols into reproducible, version-controlled workflows |
| **Literature Synthesis** | Search arXiv, PubMed, bioRxiv, medRxiv. Extract key findings and compare studies |
| **Data Pipeline Builder** | Click-to-connect pipelines: cleaning, analysis, visualisation, and export |
| **Autonomous Research** | Describe your hypothesis. Nautilus designs experiments and iterates |
| **Instrument Integration** | Connect lab instruments and sensors. Collect data automatically |
| **Reproducibility** | Every action logged, versioned, and exportable as a reproducibility package |

Everything about Nautilus is powered by Hermes underneath. When you use Nautilus, you're using Hermes — just wearing a snorkel and a lab coat.
